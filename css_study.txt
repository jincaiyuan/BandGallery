2. font size:
    1.2em means in this part size is 1.2 larger than its parent part. For example,
    body {
        font-size: 14px;
    }
    h1 {
        font-size: 150%;
    }
    h2 {
        font-size: 1.2em;
    }
    
    Tips about how to set the font size:
    a. Choose a key word (like small or medium) to be the font in the body.
    b. Use em or percentage to determine font size of other parts in body.
